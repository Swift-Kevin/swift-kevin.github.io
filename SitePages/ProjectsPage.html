Invasion of Liberty

  A 2.5D inspired remake of the arcade classic, Defender (1981). Fight endless waves of enemies as you protect the Planet of Liberty below. Dodge oncoming UFOs and projectiles in space and gain more points by taking down enemies. Try to gain the highest possible score!

Contributions: Vulkan Rendering Pipeline, Importing Custom Data, Source Control Merging, User Interfaces using 3D models

Role: Graphics Programmer / Secondary Lead

Development Time: 1 month.

Team Size: 4.

Conway's Game of Life

  A simulation where cells live, die, and repopulate the board. Originally designed by John Horton Conway, this version aims to replicate the expirament with the given rules. Made using wxWidgets and C++. This project was completed during my Project and Portfolio class where we had a month to recreate Conway's Game of Life given the original rules.

Development Time: 3 weeks.

Team Size: 1.


This was a student project in which wanted to expand the options available using wxWidgets and becoming more familiar with documentation/research. I added my own random generation seeding system and cell customizations for users to have a personalized experience.
