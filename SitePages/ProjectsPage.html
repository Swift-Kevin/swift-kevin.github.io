Conway's Game of Life

  A simulation where cells live, die, and repopulate the board. Originally designed by John Horton Conway, this version aims to replicate the expirament with the given rules. Made using wxWidgets and C++. This project was completed during my Project and Portfolio class where we had a month to recreate Conway's Game of Life given the original rules.

Development Time: 3 weeks.

Team Size: 1.
